# Aplicacion de Heroes App

### `La aplicacion contiene un listado , buscador de heroes de DC Y Marvel ,ademas se simulo el login sin backend`



### Instalación 🔧
Ejecutar

```
npm install
```

Por ultimo

```
npm start
```
